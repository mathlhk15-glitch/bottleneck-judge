# 통합 병목 투자 판정기 vFinal 100

이 프로젝트는 종목을 병목 투자 후보로 사전 판정하기 위한 HTML 기반 도구입니다.

## 주요 파일

- `index.html`: 정식 병목 투자 판정기
- `workbench.html`: 4AI 프롬프트 워크벤치 (Perplexity→Gemini→Claude→ChatGPT)
- `manual.html`: 사용 설명서
- `legacy/`: 이전 버전 보관

## 핵심 철학

좋은 회사, 병목 기업, 좋은 가격은 서로 다릅니다.

이 도구는 다음 6대 질문을 기준으로 종목을 검토합니다.

1. 없어서는 안 되는가?
2. 가격을 올릴 수 있는가?
3. 고객이 기다리는가?
4. 경쟁자가 쉽게 따라오기 어려운가?
5. 아직 시장이 과하게 반영하지 않았는가?
6. 시간이 지날수록 더 강해지는가?

## 점수 구조 (100점 환산)

| 카테고리 | 배점 |
|---------|-----|
| 사업병목성 | 30점 |
| 수요가시성 | 15점 |
| 수익성 | 20점 |
| 밸류에이션 | 15점 |
| 재무안정성 | 10점 |
| **원점수 합계** | **90점** |
| 리스크 감점 | 최대 -15점 |

원점수 90점 → 100점 환산 후 리스크 감점 적용 = 최종점수

## 등급 기준

| 등급 | 점수 기준 |
|-----|---------|
| S | 85점 이상 |
| A | 75점 이상 |
| B | 65점 이상 |
| C | 55점 이상 |
| D | 55점 미만 |

단, 사업병목성 점수에 따른 게이트 상한과 자료 신뢰도에 따른 등급 상한이 별도로 적용됩니다.

## 주요 안전장치

- **ECR cap**: EPS 대비 주가 상승이 과도할 때 밸류에이션 점수 상한 적용
- **사이클 피크 cap**: 경기민감주가 사이클 정점 이익으로 밸류에이션을 왜곡할 때 상한 적용
- **병목성 게이트**: 사업병목성 점수가 낮으면 최종 등급 상한 제한
- **자료 신뢰도 상한**: 자료가 부족하면 등급 상한 적용

## 주의

이 도구는 매수·매도 추천 도구가 아닙니다.  
점수는 모바일 간이판정 또는 사전 스크리닝 결과이며, 최종 판단은 사용자가 직접 자료를 확인해 결정해야 합니다.
